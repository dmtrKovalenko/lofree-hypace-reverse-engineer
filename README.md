# Lofree Hypace Reverse Engineering

This project is a reverse engineering result of my lofree hypace mouse. For some reason their web based HID firmware flasher doesn't support certain features on macos

![photo](./photo.png)

This investigation made it possible to flash firmware on your own bypassing all the limitations. The project contains of

1. [./PROTOCOL.md](./PROTOCOL.md) - detailed description of protocol used for flashing the firmware
2. [./macos-remap-sidebuttons-to-desktop-switch.js](./macos-remap-sidebuttons-to-desktop-switch.js) - a script that I used to remap the side buttons to desktop switching


Feel free to use the protocol to make your own firmware flashing scropt using AI to configure the mouse the way you want it to be configured.
